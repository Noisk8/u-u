Colspan

El atributo colspan en HTML se utiliza en las tablas para que una celda se extienda a través de varias columnas. Esto es útil para crear encabezados de tabla que abarcan múltiples columnas o para combinar celdas en una fila. El valor del atributo colspan especifica el número de columnas que la celda debe abarcar.

Rowspan

El atributo rowspan en HTML se utiliza en las tablas para que una celda se extienda a través de varias filas. Esto es útil para crear celdas que abarcan múltiples filas, lo que puede ser necesario para encabezados de tabla o para combinar celdas en una columna. El valor del atributo rowspan especifica el número de filas que la celda debe abarcar.
